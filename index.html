DATABOX

Um box de soluções com dados

O objetivo do Databox é apresentar alguns projetos de ciência de dados, para fins acadêmicos, profissionais e de autoaprendizagem. Aqui a regra é solucionar através de projetos utilizando Data Science, Machine Learning e Data Analysis. Todo o material pode ser consultado através do GitHub. Seja bem vindo!

Meus Projetos:

1. Projetos com Machine Learning:

  • Rossmann Sales: Forecasting Sales

    • Prevendo vendas da Rossmann, extraimos algumas hipóteses baseado no dataset disponível e fizemos algumas previsões. Os resultados são apresentados através de um Bot do Telegram.

  • Insurance All Company

    • Utilizando algorítmos de classificação calculamos a taxa churn de uma empresa de seguros veicular, através deste modelo podemos calcular a curva de ganho para diferentes porções de time de vendas.

  • Blocker Fraud Company

    • Proposta de modelo de Machine Learning para solucionar um problema de classificação binário. O objetivo é rastrear transações fraudulentas.

  • OkCupid-Date-A-Scientist

    • Através do aplicativo de relacionamento OkCupid prevemos o status de relacionamento das pessoas, auxiliando as vendas nos períodos de dia dos namorados e afins. O modelo apresenta uma acurácia de 98%.

  • Spotify Songs Popularity Prediction

    • Calculamos o sucesso das músicas do aplicativo Spotify através de suas características técnicas como: agudo, ruído, duração, energia etc. Prevemos o grau de popularidade de uma música através de uma métrica utilizada pelo próprio aplicativo.
